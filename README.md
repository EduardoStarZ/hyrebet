## Reprodução

Para a execução do código contido neste repositório, é necessário o uso do gerenciador de projetos Cargo com a versão
v1.85 do Rust ou superior, com a versão estável ou nightly.

Com a utilização do script "build.sh", o compilador irá fazer download todas as dependências do projeto e compilar todos os binários
em um diretório "bin". Ao executar o binário "hyrebet", todos os binários serão executados através dele.

Para a execução do projeto em Vue, é necessário que o pacote do NPM esteja instalado. Veja a documentação do seu sistema operacional para 
a instalação e configuração do NPM.
